Welcome to the Frame System Designer

System Designer is an editing system for creating Frame system specification documents. Frame is a markdown language that helps software architects and developers quickly and easily define state machines that capture the core system logic.

At the heart of System Designer is the Framepiler . The Framepiler is an open source transpiler written in Rust and bundled with System Designer. The Framepiler takes Frame files and transpiles them into target languages.

This version of System Designer generates the following languages:

  • Python
  • UML
  • (with more to come)
